# SOC Threat Detection Lab

## Overview

This project demonstrates the design, execution, and validation of an endpoint-focused threat detection lab built to simulate realistic attacker behaviour and analyse telemetry from a SOC analyst perspective.

---

## Project Objective

The objective of this project is to demonstrate detection engineering skills by:

- simulating attacker techniques  
- capturing endpoint telemetry with Sysmon  
- analysing behaviour using Wazuh  
- applying MITRE ATT&CK mapping  
- tuning detections for relevance  

---

## MITRE ATT&CK Coverage

- T1059.001 — PowerShell  
- T1059 — Command Interpreter  
- T1105 — Ingress Tool Transfer  

---

## Detection Workflow

Simulated behaviour → Sysmon → Wazuh → Detection → Tuning → Validation
